## Features

- Fast full-text search across all snippets
- Tag-based organization
- Code syntax highlighting for 20+ languages
- Local-first, works completely offline
- Export/Import functionality
- Dark/Light theme support

## Installation

1. Download the latest release from the [Releases](https://github.com/adrianlazariuc/codesnippetlab-releases/releases) page
2. Extract the ZIP file (Windows) or run the installer
3. Launch `CodeSnippetLab.exe`

**No installation required** - completely portable!

## Requirements

- Windows 10 or later
- macOS 10.13 or later
- Linux (Ubuntu 18.04+ or equivalent)
